# Highways 26 - Japanese Carnival Edition

This is the complete standalone implementation of the Highways 26 website, rebuilt from the ground up using React and Node.js.

## Theme
The theme for Highways 26 is **Japanese Carnival (Matsuri)**. It features vibrant visuals, celebratory elements, and a premium aesthetic that captures the essence of a festive Japanese night.

## Tech Stack
-   **Frontend**: React (TypeScript), CSS3 (Modern techniques)
-   **Backend**: Node.js, Express
-   **No Vite**: Built using standard React tools as requested.

## Project Structure
-   `/client`: React frontend application.
-   `/server`: Node.js/Express backend API.

## How to Run
1.  **Server**: `cd server && npm install && npm start`
2.  **Client**: `cd client && npm install && npm start`
